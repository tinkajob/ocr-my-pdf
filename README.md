# OCRmyPDF Scripts

A collection of scripts to **install and use OCR technology on PDFs**.

## Features
- Automates OCR installation for PDFs  
- Supports multiple languages (English & Slovenian)  
- Simple, step-by-step instructions for Windows users  

## How to Use (Windows)
1. **Download**  
   - Download the zipped folder in your preferred language (`English` or `Slovenian`).  

2. **Install Dependencies**  
   - Open the included **MS Word instruction document**.  
   - Follow the installation steps to set up all required tools and libraries.  

3. **Run the Scripts**  
   - Run OCR.bat script by following the second section at the bottom of the Word document.  
   - Script will allow you to OCR your PDFs automatically.
